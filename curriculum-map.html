📖 How to Use This Map

Four tips to get the most out of the MathMunch Curriculum Map in your classroom

1
Assign Games by Objective Code
Find the exact Cambridge (e.g., Nn3.02) or Merdeka (e.g., B.B.2) objective you're teaching, then assign the linked game pills directly to students via the MathMunch dashboard.
2
Use the Strand Filter
Select "Geometry / Geometri" from the strand dropdown to instantly see all geometry objectives and linked games across all grade levels — perfect for topic-based planning.
3
Print as a Planning Tool
Click the "Download PDF" button for a clean printable version of this map. Use it as a weekly planning reference or share it with parents and department heads.
4
Compare Curricula to Find Learning Gaps
Use "Side by Side" view to compare Cambridge and Merdeka objectives at the same level. Identify topics covered in one curriculum but not the other to enrich your teaching.
